How to set up EmailJS (free — 200 emails/month)
  1. Go to emailjs.com and create a free account
  2. Add an Email Service (Gmail, Outlook, etc.) — note the Service ID
  3. Create a receipt template with: {{to_name}} {{to_email}} {{order_id}} {{order_date}} {{order_items}} {{order_total}} {{shipping_address}}
  4. Create a shipping template with: {{to_name}} {{to_email}} {{order_id}} {{tracking_number}}
  5. Go to Account → API Keys and copy your Public Key
  6. Paste everything below and click Save
Template variables: {{to_email}} {{subject}} {{message}}
